A JavaScript implementation of Battleship


A single-player adaptation of the classic game Battleship.

Username: Password:

The player places ships within their game grid by dragging them into place. The computer then places ships in its own game grid which are hidden to the player. The player and computer then take turns selecting positions within the grid.

Upon selecting a location which is occupied by an enemy ship, the grid is marked at that location. The computer will search adjacent squares until it has sunk the ship. The game will notify the player once a ship has been sunk and when all of either the player's or the computer's ships have been sunk, the game concludes.

For more information about the game and its history, visit Wikipedia
